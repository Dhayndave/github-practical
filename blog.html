How I Made My First Website Using Just HTML and CSS

When I joined engineering, I didn't know anything about websites. A senior told me to just open Notepad, type some HTML, and open it in Chrome. That one suggestion changed everything.

I started with a basic page: a heading, a paragraph, and a button. That was it. But seeing it work in my browser was the most exciting moment. From that day I kept adding more — colours, layouts, images.

The biggest lesson? You don't need any fancy software to build a website. Just a text editor and a browser is enough to get started.

5 Simple Python Programs Every Beginner Should Try

1. Calculator — Learn how to take input and do arithmetic.

2. Guess the Number — Learn about loops and conditions (if/else).

3. Simple Quiz — Learn how to compare strings and keep score.

4. Word Counter — Learn about strings and split().

5. To-Do List in Terminal — Learn about lists and user input in a loop.

Each program teaches you one core concept. Build all five and you'll already be thinking like a programmer!

How I Study for Engineering Exams Without Getting Stressed

Step 1: Read the syllabus first. Don't waste time studying topics that aren't in the exam. I always read the unit list before opening the textbook.

Step 2: Use past papers. Solving previous year question papers is the single best way to prepare. The same type of questions keep coming back.

Step 3: Teach someone. Explaining a topic to a friend forces you to truly understand it. If you can't explain it simply, you don't understand it yet.

JavaScript in 10 Minutes

JavaScript makes your webpage interactive. Here are the 3 things I wish someone had explained to me on day one:

1. Variables — Use var or let to store data. Example: var name = "Priya";

2. Functions — A block of code you can run whenever you want. Example: function sayHello() { alert("Hello!"); }

3. Events — Things that happen when you click or type. Example: onclick="sayHello()" on a button.

That's really the core of JavaScript. Everything else builds on these three ideas!
